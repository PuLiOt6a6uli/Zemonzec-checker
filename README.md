## How to Check if You Successfully Minted

1. **Check your transaction**
   - Go to [Blockchair](https://blockchair.com/)
   - Enter your wallet address
   
2. **Find your mint transaction**
   - Copy the transaction hash where you received 0.06 USD

3. **Verify in Check.txt**
   - Open the `Checkzemon.txt` file in this repository "View RAW"
   - Use `CTRL+F` (or `CMD+F` on Mac) to search for your transaction hash

4. **Determine ownership**
   - If your transaction appears FIRST after the index number (ID) "6665", the NFT is yours! üéâ
   - If someone else's transaction appears before yours, unfortunately you missed it üî•

5. **Double-check on the website**
   - Visit [zemon.cash](https://zemon.cash/)
   - Use the search function to enter the Zemon ID
   - You'll see who was the first to mint that specific NFT
